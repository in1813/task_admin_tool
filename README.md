# タスクラベルツリー管理ツール

タスクを階層構造（ツリー）で管理できるシンプルなGUIツールです。

## 機能

- **ツリー構造でタスク管理**: 親子関係を持つタスクを階層的に表示
- **状態管理**: 各タスクに「未着手」「実行中」「完了」の状態を設定可能
- **メモ機能**: 各タスクに進捗メモを記録可能（折り畳み表示対応）
- **自動保存**: JSONファイルにデータを永続化

## セットアップ

### 1. 依存ライブラリのインストール

```bash
pip install -r requirements.txt
```

### 2. 実行

```bash
python main.py
```

## 使い方

### タスクの追加
- **ルートタスク追加**: 「＋ ルートタスク追加」ボタンをクリック
- **子タスク追加**: 親タスクを選択した状態で「＋ 子タスク追加」ボタンをクリック

### タスクの編集
1. 左側のツリーからタスクを選択
2. 右側の詳細パネルで名前・状態・メモを編集
3. 「保存」ボタンをクリック

### タスクの削除
1. 削除したいタスクを選択
2. 「削除」ボタンをクリック
3. 確認ダイアログで「はい」を選択

※子タスクがある場合は、子タスクも一緒に削除されます

### メモの折り畳み
- 「▼ メモ」ボタンをクリックするとメモ欄の表示/非表示を切り替え可能

## データ保存場所

タスクデータは `data/` にjson形式で保存されます。
